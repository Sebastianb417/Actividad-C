# 🛵 Aplicación de Domicilios en C

Aplicación de consola desarrollada en lenguaje C que simula un sistema básico de pedidos a domicilio mediante un menú interactivo.

---

## 📋 Descripción

Este programa permite al usuario interactuar con un menú de opciones para:

- Ver productos disponibles
- Agregar productos al carrito
- Visualizar el carrito de compras
- Calcular el total del pedido
- Confirmar el pedido

Se hace uso de estructuras de control como condicionales (`if`, `switch`) y ciclos (`do-while`, `for`), cumpliendo con los requisitos del ejercicio.

---

## ⚙️ Funcionalidades

- 📌 Menú interactivo
- 🛒 Carrito de compras
- 📦 Catálogo de productos (6 productos)
- 💰 Cálculo de total (incluye domicilio)
- ✅ Validación de datos (productos y cantidades)

---

## 🧱 Tecnologías utilizadas

- Lenguaje C
- Estructuras (`struct`)
- Arreglos
- Ciclos (`for`, `while`, `do-while`)
- Condicionales (`if`, `switch`)
